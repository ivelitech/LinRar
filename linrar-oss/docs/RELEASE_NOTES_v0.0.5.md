# LinRAR v0.0.5

**A small release with one honest addition: a way to say thanks.**

---

## What's new

### An optional way to support the project

LinRAR now shows a small, quiet note when you open the app, letting you
know that if it's useful to you, a donation helps keep development
going. That's the whole ask.

A few things worth being upfront about:

- **It never blocks you.** The note doesn't lock the app, doesn't pause
  you with a countdown, and doesn't stop you from doing anything —
  it's just there, and you can ignore it.
- **"Don't show this again" actually works.** Check the box once, and
  LinRAR remembers — it won't ask again on this machine.
- **There's no trial, no tiers, no unlock.** Every feature in LinRAR
  works fully whether you ever click "Support" or not, this release
  and every one after it.

There's also a small ❤ Donate link tucked into the toolbar and the
Help menu now, for anyone who wants to find it later without waiting
for the popup.

## Why add this at all

Free software still costs time to build and maintain. WinRAR's approach
— stay fully functional, just ask nicely — is a reasonable one, so
that's the shape this took: an honest ask, easy to dismiss, never
disguised as anything else.

## Everything else

No changes to core functionality in this release. Archive browsing,
creation, extraction, testing, search, and repair all work exactly as
they did in v0.0.1.

---

*Thank you for using LinRAR — and to anyone who supports it, thank you twice.*
